# Space shooter

Сборка в build:

`cmake ..`

`make`



Корабль двигается в плоскости, перпендикулярной направлению полёта.

Управление движением и камерой: WASD + мышь.

Огонь - ПКМ.

Выход - Esc.

Гипер режим - клавиша 1 (действует несколько секунд, игрок неуязвим).
